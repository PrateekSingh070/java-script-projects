# java-script-projects

Small **front-end** exercises built with **HTML**, **CSS**, and **vanilla JavaScript**—each folder is a self-contained mini app.

## Projects

| Folder | Description |
|--------|-------------|
| `Calculator/` | Basic calculator UI and logic |
| `color/` | Color-related DOM practice |
| `currency converter/` | Currency conversion UI |
| `fetching api/` | Fetch API usage |
| `palindrome/` | Palindrome check |
| `QR Generator/` | QR code generation in the browser |
| `rock paper scissor/` | Game logic |
| `stop watch/` | Timer / stopwatch |
| `tic tac toe/` | Tic-tac-toe game |
| `todo list/` | Todo list |
| `weather api/` | Weather data via an API |

## How to use

Open `index.html` in each project folder in a browser, or serve the folder with any static file server. Some projects call external APIs—use **HTTPS** or a local server if the browser blocks `file://` requests.
